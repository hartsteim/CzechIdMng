# Modal Component

Wrapped Material-UI dialog
* https://v4.mui.com/api/dialog/


## Usage

```html
<Basic.Modal show={true} onHide={() => {alert('onHide')}} bsSize="large">
  <Basic.Modal.Header text="Title" closeButton/>
  <Basic.Modal.Body>
    Body ...
  </Basic.Modal.Body>
  <Basic.Modal.Footer>
    Footer
  </Basic.Modal.Footer>
</Basic.Modal>

```
